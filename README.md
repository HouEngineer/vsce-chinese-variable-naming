# 中文变量命名

根据选中的中文右键菜单进行变量命名

![Demo](./images/demo.gif)

## Extension Settings

在VS Code 配置文件添加[百度翻译开放平台](https://fanyi-api.baidu.com/)的appid和secret

```json
    "chinese-variable-naming.appid": "your appid",
    "chinese-variable-naming.secret": "your secret",
```
